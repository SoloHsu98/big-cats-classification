Big Cats Classification

Student: Hsu Instructor: Nyein Thiha Zaw

This project aims to build an image classifier to identify different species of big cats using a Convolutional Neural Network (CNN). The model is trained, validated, and tested on a dataset containing images of various big cats, which are organized into separate directories for training, validation, and testing.

Dataset

  • Training Set: Images used to train the model.
  • Validation Set: Images used to validate the model during training.
  • Test Set: Images used to evaluate the model's performance after training.

Data Augmentation

To improve the model's generalization capability, data augmentation techniques are applied to the training images. This includes random horizontal and vertical flipping and random rotation of the images.

Model Architecture

The classifier is built using a custom CNN architecture, which consists of the following layers:

  • Data Augmentation Layer: Applies random transformations to the input images to create diverse training examples.
  • Convolutional Layers: Four convolutional layers with ReLU activation to extract features from the input images. Each convolutional layer is followed by a max-pooling layer to reduce the spatial dimensions and retain the most important features.
  • Global Average Pooling Layer: Reduces the spatial dimensions of the feature maps to a single vector.
  • Fully Connected Layer: A dense layer with 128 neurons and ReLU activation to further process the extracted features.
  • Dropout Layer: A dropout layer with a dropout rate of 0.5 for regularization to prevent overfitting.
  • Output Layer: A dense layer with 10 neurons and softmax activation to classify the images into 10 different categories.

Training

The model is compiled with the Adam optimizer, sparse categorical cross-entropy loss function, and accuracy as the evaluation metric. An EarlyStopping callback is used to monitor the validation loss and stop training when the model's performance stops improving, restoring the best weights observed during training.

Evaluation

After training, the model is evaluated on the test dataset to assess its accuracy and loss. The trained model is then saved for future use.

Results

The classifier is expected to accurately identify the species of big cats in the test images based on the features learned during training.

Conclusion

This project demonstrates how to build a robust image classifier using a custom CNN architecture. The use of data augmentation and early stopping helps in improving the model's performance and generalization capability, making it a valuable tool for classifying images of big cats.
